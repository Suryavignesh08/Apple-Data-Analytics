# 🍎 Apple Financial Dashboard (Power BI)

## 📊 Project Overview

This project presents an interactive financial dashboard built using Power BI to analyze the quarterly performance of Apple Inc. The dashboard provides insights into revenue growth, product contribution, regional performance, profitability, and cash flow.

The goal of this project is to demonstrate data analysis, visualization, and business storytelling skills using real-world financial data.

---

## 🚀 Key Features

* 📈 **Revenue Analysis**

  * Year-over-Year (YoY) growth calculation
  * Revenue trend visualization

* 📱 **Product Breakdown**

  * Contribution of iPhone, Services, Mac, iPad, and Wearables

* 🌍 **Regional Analysis**

  * Revenue distribution across Americas, Europe, China, Japan, and Asia Pacific

* 💰 **Profitability Insights**

  * Gross Margin
  * Net Income
  * Profit Margin %

* 💸 **Cash Flow Analysis**

  * Operating, Investing, and Financing cash flows

* 🎯 **Dynamic Dashboard**

  * Interactive Year slicer (2024 / 2025)
  * Dynamic title based on selected year
  * Fully responsive visuals

---

## 🛠 Tools & Technologies

* **Power BI**
* **DAX (Data Analysis Expressions)**
* **Microsoft Excel**
* Data Modeling (Star Schema)

---

## 📂 Dataset

The dataset is derived from Apple's publicly available financial statements and structured into multiple tables:

* Revenue (Product-wise)
* Region (Geography-wise)
* Profit (Key financial metrics)
* Cash Flow (Operational metrics)

---

## 🧠 Key Insights

* Apple achieved **15.6% YoY revenue growth**
* iPhone contributes **~59% of total revenue**
* Strong **profit margin (~29%)**
* Americas is the leading revenue-generating region
* Services segment is steadily growing, indicating a shift toward recurring revenue
